HalfUI 演示

簡單易用的API

HalfUI 提供直觀的API，讓您快速構建游戲界面。以下是一個簡單的示例：

#include "HalfUI.h" // 創建一個按鈕 auto button = HalfUI::Button::create("開始游戲"); button->setPosition(100, 100); button->setSize(200, 50); button->setOnClickListener([]() { startGame(); }); // 添加到UI層 uiLayer->addChild(button);
